  --->>> Link file https://github.com/langocdat/MySQL_DBA/blob/main/Scripts/OSLog.ps1
2. Run scripts:
  (powershell-Adminstrator): .\OSLog.ps1
3. Monitor:
  (powershell-Adminstrator): logman query OSLog

  (powershell-Adminstrator): logman stop OSLog
